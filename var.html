Mathematical Framework for Portfolio Optimization

Problem Formulation

The portfolio optimization problem can be formulated in two different ways, with the second approach being much more practical:

Decision Variables

$$\mathbf{w} = [w_1, w_2, \ldots, w_n]^T \in \mathbb{R}^n \text{ (portfolio weights vector)}$$ $$n = \text{number of assets/strategies}$$

Objective Functions - Two Approaches

‚ùå Formulation 1: Traditional Sharpe

$$\max S(\mathbf{w}) = \frac{\boldsymbol{\mu}^T\mathbf{w}}{\sqrt{\mathbf{w}^T\boldsymbol{\Sigma}\mathbf{w}}}$$

Issues:

  • Non-convex optimization
  • Requires VaR constraint
  • Slow to solve
  • Local minima problems

‚úÖ Formulation 2: Risk-Adjusted Return

$$\max f(\mathbf{w}) = \boldsymbol{\mu}^T\mathbf{w} - \lambda \cdot \mathbf{v}^T\mathbf{w}$$ $$= (\boldsymbol{\mu} - \lambda \mathbf{v})^T\mathbf{w}$$

Advantages:

  • Linear program!
  • No VaR constraint needed
  • Fast global optimum
  • Interpretable Œª parameter

Constraint Set

1. Budget Constraint (Always Required)

$$\sum_{i=1}^{n} w_i = 1 \quad \text{or} \quad \mathbf{1}^T\mathbf{w} = 1$$

2. VaR Constraint (ELIMINATED in Formulation 2!)

$$\text{VaR}_{\max} - \mathbf{v}^T\mathbf{w} \geq 0$$

This constraint is no longer needed! The Œª parameter in the objective function controls risk preference instead.

3. Concentration Constraint

$$w_i \leq w_{\max} \quad \forall i \in \{1,\ldots,n\}$$

4. Non-negativity Constraints

$$w_i \geq 0 \quad \forall i \in \{1,\ldots,n\}$$

Complete Mathematical Programs

‚ùå Traditional Approach

$$\begin{align} \text{minimize} \quad & -\frac{\boldsymbol{\mu}^T\mathbf{w}}{\sqrt{\mathbf{w}^T\boldsymbol{\Sigma}\mathbf{w}}} \\ \text{subject to:} \quad & \mathbf{1}^T\mathbf{w} = 1 \\ & \text{VaR}_{\max} - \mathbf{v}^T\mathbf{w} \geq 0 \\ & 0 \leq w_i \leq w_{\max} \quad \forall i \end{align}$$

Type: Non-convex QCQP

‚úÖ Preferred Approach

$\begin{align} \text{minimize} \quad & -(\boldsymbol{\mu} - \lambda \mathbf{v})^T\mathbf{w} \\ \text{subject to:} \quad & \mathbf{1}^T\mathbf{w} = 1 \\ & 0 \leq w_i \leq w_{\max} \quad \forall i \end{align}$

Type: Linear Program

Note: Negative sign because solvers minimize, but we want to maximize expected return minus risk penalty.

Risk Model Integration

Linear VaR Aggregation

Both formulations use the same VaR approximation:

$$\text{Portfolio VaR} \approx \sum_i w_i \cdot \text{VaR}_i = \mathbf{v}^T\mathbf{w}$$

Key Insight: In the preferred formulation, we optimize:

$$\text{Expected Return} - \lambda \times \text{Expected VaR}$$
