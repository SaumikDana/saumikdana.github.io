performance_metrics.py - Statistical Analysis

Portfolio-level analytics and investment performance metrics

# Rolling window calculations
def sharpe(x, window, risk_free_rate=0, period=252):
    excess_returns = x - risk_free_rate
    r = excess_returns.rolling(window=window)
    m = r.mean()
    s = r.std(ddof=0)
    return (m / s) * np.sqrt(period)

# Statistical methods on Series
_sum_.at[ret_ts, 'skew'] = _returns.skew()
_sum_.at[ret_ts, 'badday'] = _returns.min()

# Conditional filtering and aggregation
_sum_.at[ret_ts, 'pct_days_on'] = (_returns != 0).sum() / _returns.shape[0]
_sum_.at[ret_ts, 'sharpe_on'] = sharpe(_returns[_returns != 0], len(_returns[_returns != 0]))[-1]

Pandas Patterns & Techniques

1. Multi-Level Dictionary → DataFrame Pattern

# Start: {option_id: DataFrame} dictionary
dict_options = {'call_A': df1, 'put_B': df2, ...}

# Transform: Column-wise aggregation
df_combined = pd.DataFrame({o: opt['pnl'] for o, opt in dict_options.items()})

2. Time Series Analysis Pattern

# Cumulative calculations
perf_cum = np.cumprod(ret_series + 1) - 1
hwm = np.maximum.accumulate(perf_cum)

# Rolling statistics
r = x.rolling(window=window)
zscore = (x - r.mean().shift(1)) / r.std(ddof=0).shift(1)

3. Boolean Masking for Complex Conditions

# Multiple condition masks
mask = (df.index >= exclude_date) & (df.index <= exdate)

# Conditional value assignment

4. Index-Based Operations

# Index alignment in merges

Memory Efficiency Considerations

In-place Operations

Uses inplace=True where possible to avoid DataFrame copying and reduce memory usage

Conditional Assignments

Leverages .loc[] instead of creating new DataFrames for better performance

Vectorized Operations

Avoids Python loops by using pandas vectorization for mathematical operations

Selective Processing

Only processes required columns to minimize computational overhead

Key Takeaway

The pandas approach emphasizes time series analysis, conditional logic through boolean indexing, and efficient aggregation patterns - all critical for financial data processing and options P&L analysis.
