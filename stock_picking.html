The Stock Analysis Tool is a comprehensive Python-based program designed to analyze stock options and track stock prices. It begins by scraping earnings data from Yahoo Finance, followed by data preparation where it processes earnings tables and converts them into a sorted DataFrame. The tool then performs an in-depth options analysis, including filtering tickers and plotting Greeks using the Black-Scholes model. It calculates call and put prices and their respective Greeks (Delta, Gamma, Theta, Vega, and Rho). For visualization, it plots both current stock prices and historical data. Additionally, it includes utility functions for fetching stock prices, extracting data from URLs, and printing options data, making it a versatile tool for financial analysis and decision-making.
