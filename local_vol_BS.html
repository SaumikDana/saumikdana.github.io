Mathematical Implementation

Local Volatility Black-Scholes PDE with SVI Surface Calibration

I. The Local Volatility Black-Scholes PDE

Governing Equation

$$\frac{\partial V}{\partial \tau} = \frac{1}{2}\sigma^2(S,\tau)S^2 \frac{\partial^2 V}{\partial S^2} + (r-q)S \frac{\partial V}{\partial S} - rV$$
(1)

Variable Definitions:

\(V(S,\tau)\): Option value as function of spot S and time-to-expiry τ
\(\sigma(S,\tau)\): Local volatility surface (space and time dependent)
\(r\): Risk-free interest rate
\(q\): Continuous dividend yield
\(\tau = T - t\): Time to expiration (backward time)

Coordinate Transformation to Moneyness Space

$$m = \frac{S}{K}, \quad u(m,\tau) = \frac{V(S,\tau)}{K}$$
(2)

PDE in Moneyness Coordinates

$$\frac{\partial u}{\partial \tau} = \frac{1}{2}\sigma^2(m,\tau)m^2 \frac{\partial^2 u}{\partial m^2} + \left((r-q) - \frac{1}{2}\sigma^2(m,\tau)\right)m \frac{\partial u}{\partial m} - ru$$
(3)
Transformation Benefits:
• Dimensionless coordinates independent of strike level
• Natural boundary conditions: \(u(0,\tau) = 0\), \(u(\infty,\tau) = m - e^{-(r-q)\tau}\)
• Improved numerical stability for wide strike ranges

Initial and Boundary Conditions

$$u(m,0) = \max(m-1, 0) \quad \text{(Call payoff at expiration)}$$
(4)
$$u(0,\tau) = 0 \quad \text{(Deep OTM boundary)}$$
(5)
$$u(m_{\text{max}},\tau) = m_{\text{max}} - e^{-(r-q)\tau} \quad \text{(Deep ITM boundary)}$$
(6)

II. SVI Volatility Surface Calibration

SVI Parameterization

$$w(k) = a + b\left[\rho(k-m) + \sqrt{(k-m)^2 + \sigma^2}\right]$$
(7)

SVI Parameters:

\(k = \ln(K/F)\): Log forward moneyness
\(w = \sigma^2T\): Total implied variance
\(a\): Variance level (controls ATM variance)
\(b \geq 0\): Variance slope (controls skew intensity)
\(\rho \in [-1,1]\): Skew parameter
\(m\): Translation parameter (shifts minimum)
\(\sigma > 0\): Variance curvature parameter

Calibration Objective Function

$$\text{minimize: } \sum_i \left(\frac{w_{\text{market},i} - w_{\text{SVI}}(k_i)}{w_{\text{market},i}}\right)^2$$
(8)
Subject to no-arbitrage constraint:
$$a + b\sigma\sqrt{1-\rho^2} \geq 0 \quad \text{(ensures non-negative variance)}$$

SVI Calibration Algorithm

Extract market total variance: \(w_{\text{market}} = \text{IV}^2 \times T\)
Generate intelligent initial guess from data extrema
Apply SLSQP optimization with bounds: \(b \geq 0\), \(\rho \in [-1,1]\), \(\sigma > 0\)
Enforce no-arbitrage constraint: \(a + b\sigma\sqrt{1-\rho^2} \geq 0\)
Validate convergence and parameter reasonableness

III. Crank-Nicolson Finite Difference Scheme

Discretization

$$\text{Grid: } m_i = i \cdot \Delta m, \, i = 0,1,\ldots,M \quad \tau^n = n \cdot \Delta\tau, \, n = 0,1,\ldots,N$$
(9)
$$u_i^n \equiv u(m_i, \tau^n)$$
(10)

Finite Difference Operators

$$\frac{\partial u}{\partial m}\bigg|_i \approx \frac{u_{i+1} - u_{i-1}}{2\Delta m}$$
(11)
$$\frac{\partial^2 u}{\partial m^2}\bigg|_i \approx \frac{u_{i+1} - 2u_i + u_{i-1}}{(\Delta m)^2}$$
(12)

Crank-Nicolson Formulation

$$\frac{u_i^{n+1} - u_i^n}{\Delta\tau} = \frac{1}{2}\left[L[u^{n+1}]_i + L[u^n]_i\right]$$
(13)

Where \(L\) is the spatial differential operator:

$$L[u]_i = \frac{1}{2}\sigma_i^2 m_i^2 \frac{\partial^2 u}{\partial m^2}\bigg|_i + \left((r-q) - \frac{1}{2}\sigma_i^2\right)m_i \frac{\partial u}{\partial m}\bigg|_i - ru_i$$
(14)

Matrix Form: \(Au^{(n+1)} = Bu^n\)

Coefficient Definitions:
$$\alpha_i = \frac{1}{2}\frac{\sigma_i^2 m_i^2}{(\Delta m)^2} \quad \beta_i = \frac{((r-q) - \frac{1}{2}\sigma_i^2)m_i}{2\Delta m} \quad \gamma_i = r$$
(15)
Matrix A (implicit side):
A[i,i-1] = -½Δτ(α_i - β_i) A[i,i] = 1 + ½Δτ(2α_i + γ_i) A[i,i+1] = -½Δτ(α_i + β_i)
Matrix B (explicit side):
B[i,i-1] = ½Δτ(α_i - β_i) B[i,i] = 1 - ½Δτ(2α_i + γ_i) B[i,i+1] = ½Δτ(α_i + β_i)

Stability and Convergence Analysis

Unconditional Stability: Crank-Nicolson scheme is unconditionally stable for the diffusion equation, allowing larger time steps without numerical instability.
Convergence Properties:
• Second-order accuracy in both space \((\Delta m)^2\) and time \((\Delta\tau)^2\)
• Truncation error: \(O((\Delta m)^2 + (\Delta\tau)^2)\)
• Optimal for smooth solutions with moderate computational cost

Validation Methodology

The theoretical option prices \(V_{\text{theory}}\) are systematically compared against market bid-ask spreads to quantify pricing accuracy and identify potential arbitrage opportunities:

$$\text{Pricing Accuracy} = \frac{|V_{\text{theory}} - V_{\text{mid}}|}{V_{\text{mid}}}$$
$$\text{Within Spread} = \mathbb{1}\{V_{\text{bid}} \leq V_{\text{theory}} \leq V_{\text{ask}}\}$$
Statistical Framework: Hit rate analysis, spread distribution characterization, and pricing error decomposition across moneyness and maturity dimensions.
